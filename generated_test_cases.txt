Story ID: SCRUM-6
### User Story: User Login

**As a** registered user, **I want** to log in to my account **so that** I can access my personalized dashboard.

#### Test Case 1: Successful Login with Valid Credentials

**Preconditions:** The user has an active account with valid credentials.

**Steps:**
1. Navigate to the login page of the application.
2. Enter a valid username in the "Username" field.
3. Enter the correct password in the "Password" field.
4. Click on the "Login" button.
5. Wait for the application to process the login request.
6. Observe the resulting page.

**Expected Result:** 
- User is redirected to their personalized dashboard.
- A success message appears (e.g., "Welcome, [User's Name]!").
- The user's account information is displayed correctly.

---
