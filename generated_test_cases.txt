### Test Case 1: Successful User Login

**Title**: Successful User Login

**Description**: Validate that a user can log in successfully with valid credentials.

**Preconditions**: 
- User has a valid email and password.
- User is on the login page.

**Steps**:
1. Navigate to the login page.
2. Enter the valid email in the email input field.
3. Enter the valid password in the password input field.
4. Click the "Login" button.

**Expected Result**: The user is directed to the dashboard/homepage, and a welcome message is displayed.

**Postconditions**: User session is created, and the user is logged in.
