### User Story: User Login
**As a** user,  
**I want** to log in to my account,  
**So that** I can access my personalized dashboard.

### Test Case 1: Successful User Login

**Test Case ID**: TC001  
**Test Case Name**: Successful User Login  
**Preconditions**: User has a registered account with valid credentials.

#### Steps:
1. Open the application or website.
2. Click on the "Login" button.
3. Enter a valid username in the "Username" field.
4. Enter a valid password in the "Password" field.
5. Click on the "Submit" button.
6. Observe the system response.

#### Expected Result:
- User is redirected to their personalized dashboard.
- A welcome message is displayed (e.g., "Welcome back, [Username]!").
